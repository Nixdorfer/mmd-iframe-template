欢迎来到Nix的iframe自动注入模板仓库

# 仓库结构

该仓库由多个不同的项目组成 其中

- [template](./template)文件夹中存储着可以通过[KEY:k1=v1;kx=vx]被正则捕获并自定义的模板型项目

- [instance](./instance)文件夹中存储着为某项目特化形成的项目

- [test](./test)文件夹中存储着用于测试新功能的文件 一般而言 您不会需要查看该文件

# 项目结构

每个模板型项目都包含以下三个文件

- README.md 为该项目的可用参数说明

- inject.html 为该项目的正则替换内容

- index.html 为该项目的模板 一般而言 您不会需要查看该文件
